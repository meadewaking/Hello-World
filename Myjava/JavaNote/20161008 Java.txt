1. 数组：在面向对象程序设计中，数组是对象。
2. 数组引用的声明语法：
   数据类型 [] 数组引用名; 
或 数据类型 数组引用名[];
在声明数组引用时不需要在中括号中确定数组大小。
3. 创建数组对象的语法：
   new 数据类型[数组大小]; 
创建数组时不是调用构造方法。
4. 访问数组元素的方式与C语言相同，遍历数组元素时可以使用数组的length属性获取数组大小。
5. 遍历集合元素的循环：
for (元素数据类型 临时变量名 : 集合对象名) {

}
这种循环仅适合读取操作，不适合做写入操作。
之后会讲到传值和传引用的问题。
6. Object类
（1）Object类是所有类的祖先类，创建Java类时未指定基类则默认基类为Object；
（2）Object类有以下一些方法：
toString(),clone(),equals(),hashCode(),finalize(),
getClass(),notify(),notifyAll(),wait(),
其中5个可重写，其它的方法不可重写的final方法；
（3）toString()方法，是将对象转化为字符串信息，默认功能格式：对象类型@编号；该方法会被自动调用；
（4）hashCode()方法，获取对象在JVM中的唯一编号，该编号称为哈希码；
（5）getClass()方法，获取对象的类信息；
7. final关键字
（1）修饰类，该类不能做基类；
（2）修饰方法，该方法不能被重写；
（3）修饰字段，该字段是常量。
